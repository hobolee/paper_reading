从“更强模型”转向“更可靠系统”：AgentCompass 和 MEDA 都不是单纯提高单任务分数，而是在重构评测或发现流程本身，关键词是解耦、复现、容错、约束注入与可诊断性。 多智能体/工具增强正在进入真实异质数据场景：UrbanAgent 把多模态城市画像从静态表示学习推向证据冲突协调、主动补证和闭环判断，显示 agent 范式开始对接遥感、POI、文本、3D 建筑这类不一致现实数据。 鲁棒性评测从英语中心走向低资源语言与扰动测试：GSM-Plus-BN 的重点不是再做一个数学数据集，而是通过 Bengali 场景下的语义扰动检测模型是否真的在推理。 环境研究聚焦“制度/地理变化如何重写碳与土地系统”：Aral Sea 干涸对应区域碳库存重构，Amazon Soy Moratorium 则对应供应链治理变化对毁林风险的再分配，两者都强调系统级后果。 可持续化学正在寻找绕开传统催化瓶颈的新物理化学路径：微液滴介导废塑料转化的核心不是更复杂催化剂，而是利用界面自由基化学实现温和、可扩展、对混合底物更友好的氧化断链。 高影响生命科学论文持续从相关性走向可操纵机制：裸鼹鼠 queen odour 工作和 in vivo antibody–ADC click 都体现出一个共同趋势，即把现象压缩到具体分子或模块化干预单元，再通过操纵实验验证因果作用。

可做研究点

复现并系统比较统一 agent 评测基础设施的最小原型

AgentCompass 指向的痛点非常普遍：同一类 agent 在不同 benchmark 上往往因 harness 和 environment 差异而无法公平比较。做一个最小复现能直接服务本组后续所有 agent 项目。

第一步：选 2 个交互式 benchmark 和 1 个静态 benchmark，手工定义 Benchmark/Harness/Environment 三层接口，对同一个基线 agent 跑通统一日志记录与 trajectory 保存，再检查哪些 benchmark 特定逻辑仍无法抽象。

风险：论文可能在工程细节和运行时机制上依赖较复杂实现；若只凭摘要复现，容易低估 environment 封装和容错调度的工作量。

做一个“轨迹级失败模式”小综述并配套标注模板

多步 agent 任务里，最终分数无法区分规划错误、工具调用错误、环境状态误解和 reward hacking。即使不完整复现 AgentCompass，也可以先把 trajectory analysis 变成可复用研究资产。

第一步：收集 20-30 条已有 agent 运行轨迹，制定最小标签集，如任务理解错误、工具选择错误、参数错误、循环/停滞、利用评测漏洞、环境异常恢复失败，并进行双人标注一致性检查。

风险：不同 benchmark 的错误类型异质性很强，统一标签体系可能过粗或过细；若轨迹记录不完整，很多失败原因无法可靠归因。

检查低资源语言数学推理中的“扰动敏感性是否高于翻译敏感性”

GSM-Plus-BN 提供了一个很好的切入问题：模型在 Bengali 上变差，到底主要来自语言迁移本身，还是来自扰动后对表面模式的依赖。这个问题对跨语言鲁棒性评测很关键。

第一步：从公开可得的 GSM 类英语样本中选一小批题，构造人工核验的 Bengali 翻译版与 2-3 类语义保持扰动版，比较同一模型在英文原题、孟加拉语直译题、孟加拉语扰动题上的性能差分。

风险：如果没有高质量人工翻译与答案核验，语言因素和数据噪声会混在一起；不同模型对 Bengali 支持度差异也会放大解释难度。

把 MEDA 式“知识约束+符号回归”框架先在一个经典 ODE 基准上做剥离实验

这类系统最值得验证的不是端到端是否有效，而是 LLM 提供的变量约束、机理约束、候选结构建议各自有没有独立增益。做剥离实验更容易得出稳健结论。

第一步：选 1-2 个经典生物动力学系统，建立四组对照：纯符号回归、符号回归+变量约束、符号回归+机理约束、完整 agent 流程；比较方程恢复率、参数可辨识性和带噪声时的稳定性。

风险：若基准过简单，LLM 约束的价值体现不出来；若基准过复杂，符号回归本身可能成为主要瓶颈，导致无法判断增益来源。

针对多模态城市画像任务，设计“证据冲突”基准切片而非只看总体精度

UrbanAgent 的核心主张是处理模态不一致和未见区域，但这类能力常被总体平均指标掩盖。构建冲突切片能更直接检验多智能体协作是否真有必要。

第一步：在现有城市区域数据中筛选或人工构造一组样本，使卫星、POI、文本、3D 建筑给出不一致信号，然后比较单模态模型、早融合模型和简单多 agent 协作原型在这些样本上的表现。

风险：证据冲突样本的定义可能主观，且不同模态质量不均会让“冲突”与“噪声”难以区分。

围绕“环境变化如何改写碳账本”做一篇跨案例短综述草稿

Aral Sea 碳库存重塑与 Amazon Soy Moratorium 的土地治理变化，虽然对象不同，但都属于地理过程或制度变化重写区域碳与土地系统的案例。把这类研究串起来有助于形成新的选题框架。

第一步：先整理 8-12 篇与内陆水体变化、农产品零毁林政策、区域碳清单重估相关的论文，按“驱动因素—观测手段—碳/土地后果—不确定性来源”做表格化比较。

风险：仅凭题目和摘要容易把不同尺度问题强行并置；如果缺少方法细节，综述可能停留在概念拼接而非可操作比较。

一句话

这篇论文提出 AgentCompass，一个面向 LLM-based agent 的统一评测基础设施，目标是把 Benchmark、Harness 和 Environment 三类组件解耦，用于支持跨 20+ 基准、5 类能力维度的代理评估。摘要显示其重点任务不是提出新代理算法，而是解决当前 agent evaluation pipeline 碎片化、强耦合、难复现的问题，并提供异步容错运行与轨迹分析能力。

1问题

论文要解决的具体问题是：当前 LLM-based agent 的评测流程在 benchmark 定义、执行 harness、环境封装和结果分析上高度碎片化且强耦合，导致复现困难、跨 benchmark 迁移成本高、工程重复严重，也不利于定位多步交互任务中的细粒度失败模式。作者因此尝试回答：能否构建一个统一、可扩展、可复现的 agent evaluation infrastructure，使不同任务、环境与执行逻辑可以模块化组合，并支撑大规模稳定运行与轨迹级诊断？

2方法

Abstract Eusociality is exceptionally rare in mammals, and its proximate mechanisms remain poorly understood. Naked mole-rats provide a striking example, with reproduction restricted to a single queen, while all other females remain infertile 1 . Here we identify a chemical signal in queens that can regulate this reproductive hierarchy. Isopropyl myristate is detected by peripheral and central olfactory neurons and elicits avoidance in high-ranking animals. Exposure alters levels of prolactin and progesterone in non-breeders to suppress reproduction. Daily addition of isopropyl myristate to a queenless colony prevented queen succession, whereas withdrawal triggered aggression and reproductive competition. Isopropyl myristate was also detected in breeding females of several Fukomys mole-rat species, but reaches higher levels in naked mole-rats, paralleling their extreme reproductive skew. These findings reveal a chemical cue that can mediate reproductive suppression in a mammal, linking insect and mammalian eusociality.
