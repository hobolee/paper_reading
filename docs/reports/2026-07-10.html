今天最值得优先阅读的主线，是“AI 从回答问题走向执行研究与工程流程”。其中优先级最高的是 Science 的 Biomni：它把生物医学研究任务拆成可执行动作空间，并用 LLM + 检索增强规划 + 代码执行去动态组合数据库、工具、协议和实验流程，代表了通用科研代理从信息整合走向流程自治的明显跃迁。与之相呼应，软件工程 agent 评测论文和 PA-SciML 论文分别从两个关键侧面补足这一趋势：前者讨论怎样在真实开发环境里可靠评估 agent，而不是只看受污染的静态基准；后者强调 agent 自动发现模型时必须把“物理可审计的验收规则”前置，而不能只依赖验证误差。这三篇合起来构成一个很清晰的研究闭环：代理能否执行、执行得是否真实、以及结果是否可审计。 第二条高价值方向是“把基础模型嵌入真实环境并压缩到可落地形态”。GemNav 说明冻结的多模态大模型未必只能做通用感知/对话，也可以通过离散 token 化控制接口和轻量 LoRA 微调，直接承担真实机器人视觉导航策略，而且在小数据条件下追求未见环境零样本迁移。这和 Biomni 在生物医学环境中组织工具调用，本质上都在回答同一个问题：基础模型如何不靠大规模任务专用重训，而是通过环境接口设计实现跨任务能力外化。 第三条值得跟踪的是“对模型内部机制与外部行为的双重解释”。机械可解释性综述并非单一新方法论文，但它把 circuits、稀疏特征、因果干预、steering 和符号推理串成一条方法链，对当前 agent 安全、审计与可控性很有背景价值；而 Nature 那篇“LLM 可预测社会科学实验结果”虽然当前信息不足，但题目本身提示了一个更广的问题：语言模型是否已经具备某种可迁移的人类行为先验。若与机制解释研究结合，后续可能形成“行为预测能力来自什么内部表征”的新问题。 非 AI 论文里，几篇顶刊工作都围绕“复杂环境中的定向与适应”展开，彼此之间意外地形成了方法学呼应。植物根通过局部 pH 梯度避开腐解区，提出了新的 saprotropism；高海拔叶耳鼠论文把极端环境适应从低氧扩展到代谢与解毒；神经元极性研究则把轴突选择归因于胞体发起的内源性细胞骨架振荡；海底扩张事件论文用原位 seismogeodesy 重建张裂、岩浆输运和喷发的完整时序。它们虽然不属于同一领域，但共同特征是：都在从高维动态环境中识别“可指令性信号”，并用多尺度观测把信号与最终形态/功能结果连接起来。 如果按阅读优先级排序：第一梯队建议读 Biomni、软件工程 agent 评测、PA-SciML；它们直接关系到当前 AI agent 研究的能力边界、评价边界和可信边界。第二梯队读 GemNav 与机械可解释性综述，前者看落地范式，后者补机制与安全背景。第三梯队可按个人兴趣选择领域顶刊：做生物与复杂系统的优先看根趋避和神经元极性；做地球系统观测的看海底扩张事件；做进化生理的看高海拔鼠适应。至于“LLM 预测社会科学实验”这篇，由于现有信息太少，适合先保留关注，待拿到摘要或正文后再判断其证据强度。

主题线索

从“会回答”到“会执行”：科研与工程代理开始显式依赖环境接口、动作空间和动态工作流组合，而不只是文本生成。 从“高分模型”到“可验收系统”：agent 评估正从静态任务分数转向污染控制、轨迹记录、物理规则核验和现实环境中的失败模式分析。 基础模型的轻适配落地：冻结大模型主体、通过 LoRA 或离散接口适配到机器人导航等具身任务，成为低数据部署的一条现实路线。 解释对象正在升级：不仅要解释模型输出，还要解释内部电路、稀疏特征与可控干预如何支撑外部行为与推理能力。 复杂系统中的“局部信号—全局行为”机制：从根系避开腐解区、神经元指定轴突，到洋中脊扩张和高海拔适应，今天多篇工作都在解析局部梯度或内部振荡如何驱动宏观方向性结果。

可做研究点

整理并复现“agentic environment”的最小实现框架，对比生物医学代理、软件工程代理和机器人代理在动作空间表示上的共性与差异。

Biomni、软件工程 agent 评测和 GemNav 虽然任务不同，但都依赖“环境接口设计”而非单纯换更大模型。先抽象出动作、观察、记忆、规划、执行、审计六个模块，有助于形成跨领域可复用的 agent 系统学框架。

第一步：先做一页对照表，只基于现有摘要和可获得补充材料，逐项记录三类系统的输入、可调用工具、输出形式、反馈信号和评估方式。

风险：不同论文粒度差异很大，且 Biomni 与 GemNav 一个偏工具调用一个偏策略控制，强行对齐可能过于抽象。

做一个“小型 verification-first agent benchmark”原型，在公开科学机器学习或工程回归任务上比较“只看验证误差”与“误差+规则核验”选模结果。

PA-SciML 的核心价值在于说明自动模型发现如果没有外部约束，容易选出数值上看似好但物理上失真的模型。这一思想可外推到更多代理选模场景。

第一步：选一个有明确约束的公开任务，先人工写出2到4条 machine-checkable 规则，再对若干候选模型分别按验证误差和复合评分排序，检查排名是否改变。

风险：若任务本身约束定义不清或规则太弱，可能看不到与纯误差排序的显著差异。

如果科研代理从论文、协议和数据库中检索答案，训练-测试污染会比普通问答更隐蔽。软件工程评测论文提供了一个很好的切入点。

对 GemNav 的“离散 token 控制”做一个小型复现实验，检验离散化输出相对连续回归头在小数据下是否更稳。

这篇工作挑战了导航必须用专门控制头的常见假设。如果离散 token 化确实有利于利用预训练语言头，那对具身智能接口设计很重要。

第一步：选一个公开视觉导航或 waypoint 数据集，搭建一个最小对照：同一视觉骨干下比较离散 token 预测与连续坐标回归。

1问题

论文试图解决的具体问题是：面对生物医学研究中由数据库检索、分析软件调用、协议查阅、代码处理和实验操作组成的碎片化流程，能否构建一个通用AI代理，使其不依赖任务专用模板或调参，就能自主完成从信息检索、计划制定到分析执行与实验流程生成的一系列研究任务。更具体地说，作者关心的是如何把生物医学研究中的“可执行动作”系统化表示出来，并让代理在因果基因优先级排序、药物再利用、罕见病诊断、微生物组分析和分子克隆等任务之间迁移泛化。

2方法

根据摘要，方法可能包括两部分。其一是环境构建：使用一个action-discovery agent扫描数千篇文献，从25个生物医学领域中抽取工具、数据库和protocol，形成统一的agentic environment；这一步的对象很具体，可能涉及生信分析工具、医学知识库、分子设计资源、实验protocol库和设备控制接口。其二是通用代理执行框架：以大语言模型作为推理核心，结合检索增强规划来决定下一步该调用哪些资源，再通过代码式执行连接外部工具和数据处理流程，动态组合工作流而不是套固定模板。评估上，作者可能构建了多任务benchmark，比较Biomni在不同任务中的成功率、准确率、召回率、候选排序质量或专家评价，并辅以真实案例展示其在多模态数据解释、蛋白稳定性优化、湿实验仪器控制和实验协议生成中的端到端能力。

局限/风险

Visual navigation policies built on large pretrained models have so far followed a common recipe: a dedicated visual encoder, a bespoke action head, and training on thousands of hours of cross-embodiment datasets. We ask whether this recipe is necessary. In this paper, we introduce GemNav, a visual robot navigation policy that adapts a frozen Multimodal Large Language Model (MLLM) for short-to-medium horizon waypoint navigation using Low-Rank Adaptation (LoRA) on the language tower alone, with no auxiliary visual encoder and no continuous regression head. Waypoints and categorical navigation signals share a single discrete token vocabulary generated by the language-model head, and a soft-decoded auxiliary loss recovers the metric structure that pure cross-entropy training discards. Conditioning on short image histories improves offline metrics but yields no robot benefit, pointing to a ceiling on what temporal context adds once pretrained vision features are in place. These results indicate that discrete-token adaptation of frozen MLLMs can provide a data-efficient, deployable alternative for foundation model robot navigation.

Abstract Neurons acquire polarity by specifying one neurite as the axon, whereas the others become dendrites. But how this fundamental asymmetry is established remains unclear 1 . Neuronal polarization has been thought to rely primarily on growth cones that sense external cues 2 . Here we show that growth cones alone do not direct this process and that the soma acts as a central organizer of neuronal polarization. Using live imaging and genetic loss-of-function approaches in vivo, combined with optogenetic control and local cytoskeletal perturbations in cultured neurons, we uncover a soma-initiated oscillatory program that primes axon selection. Periodic actin branching that depends on the actin-related protein 2/3 (ARP2/3) complex at the soma remodels a global actomyosin network, thereby generating an actin wave that retracts neurites before propagating into a single neurite tip. Exposure to this wave relaxes local actomyosin contractility, which drives a transient microtubule-based protrusion and biases this neurite towards axon fate. As the cell exits this oscillatory stage, this neurite can overcome global inhibition and extend independently of ARP2/3, whereas actomyosin activity suppresses axon formation in the remaining neurites so that they subsequently become dendrites. This soma-driven mechanism ensures the emergence of a single axon independent of environmental cues and underpins the unidirectional information flow in neuronal circuits.
